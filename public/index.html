    Command Overview

    All supported commands. You can run them in chat or from the command bar in Controls. If you skip Bot., the dashboard adds it automatically.

    Basics + Navigation

    • Bot.testQuick health check response.
    • Bot.comeBot moves to the player who sent the command.
    • Bot.goto <player>Pathfind to a visible player and keep tracking them.
    • Bot.goto <x> <y> <z>Pathfind to exact coordinates.
    • Bot.goto.nearestGo to the nearest player found.
    • Bot.follow <player>Continuously follow a visible player.
    • Bot.follow.stopStop any active follow behavior.

    Combat + Guard

    • Bot.attack <player>Attack a visible player target.
    • Bot.pvp.stopStop current PvP attack actions.
    • Bot.guard <x> <y> <z>Guard a specific coordinate area.
    • Bot.guard.hereGuard the bot's current position.
    • Bot.guard.stopDisable guard mode.
    • Bot.selfdefenseToggle automatic self-defense on damage.
    • Bot.selfdefense.onEnable self-defense.
    • Bot.selfdefense.offDisable self-defense.
    • Bot.selfdefense.statusShow whether self-defense is enabled.

    Crafting

    • Bot.craft <itemId> [count]Check inventory, find a nearby crafting table when required, and craft the requested item.
    • Bot.smelt <itemId> [count] [station]Use furnace-like stations for smelting recipes. Station can be auto, furnace, smoker, or blast_furnace.

    Mining + Resources

    • Bot.collect <blockType> <count>Collect a block type a fixed number of times.
    • Bot.mine <blockType>Start continuous mining loop for one block type.
    • Bot.miner.stopStop the active mining loop.
    • Bot.emptyMove to a nearby chest and deposit inventory.

    Food + Chat Behavior

    • Bot.autoEatEnable auto-eat behavior.
    • Bot.autoEat.stopDisable auto-eat behavior.
    • Bot.eatForce one immediate eat attempt.
    • Bot.silentToggle silent mode for bot chat output.
    • Bot.silent.onEnable silent mode.
    • Bot.silent.offDisable silent mode.
    • Bot.silent.statusShow whether silent mode is enabled.

    Server + Starter Bot Settings

    Saved to disk. Restart bots to apply connection/auth changes. Command autocomplete suggestions use your configured MC version.
